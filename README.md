# Coral-Bleaching-Predictor-v1
A Coral Reef Degradation Prediction Prototype (v1) I made in 2024-25 as my Project

# Brief:
This project was made by scraping data from various online sources. Through that data, it uses Machine Learning algorithms to find connections and predict the amount of coral bleaching in particular areas based on data that we give. Quite a few emails were sent across a few companies as enquiry for research towards the project and replies were received, which helped my team in making it! Raw data was too heavy so it has not been uploaded. Sources have been cited for reference.

# Constraints
Data used, was combined from various websites which was tough to map across different sources

# Results
It was found to be fairly accurate considering the size of the data given

# Limitations
Availability of mapped data
